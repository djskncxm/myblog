对于环境来说绝对是 Linux 环境，虚拟机，WSL，实体机等等，Windows 就别想了
我们来分两条线来进行寻找，第一个就是小米 8 第二个是一加 10Pro，小米 8 是基础路线，标示我们需要什么，一加 10Pro 作为目标，从 0 开始寻找一个设备的所需内容进行编译

首先就是 lineageOS 的源码了，我选择的版本是 19.1 安卓 12(API 32)，以后改机等等也选择这个版本

```bash
repo init -u https://github.com/LineageOS/android.git -b lineage-19.1
repo init -u https://mirrors.tuna.tsinghua.edu.cn/git/lineageOS/LineageOS/android.git -b lineage-19.1
```

然后来配置一些源，lineage 和 AOSP 的源，如果外网就太慢了

```bash
  <remote  name="github"
           fetch=".."
           review="review.lineageos.org" />
改成
  <remote  name="github"
           fetch="https://github.com/" />
  <remote  name="lineage"
           fetch="https://mirrors.tuna.tsinghua.edu.cn/git/lineageOS/"
           review="review.lineageos.org" />

将
  <remote  name="aosp"
           fetch="https://android.googlesource.com"
改成
  <remote  name="aosp"
           fetch="https://mirrors.tuna.tsinghua.edu.cn/git/AOSP"

将
  <default revision="..."
           remote="github"
改成
  <default revision="..."
           remote="lineage"
```

最终是这样的

```
<remote  name="github"
       fetch="https://github.com/" />
<remote  name="lineage"
       fetch="https://mirrors.tuna.tsinghua.edu.cn/git/lineageOS/"
       review="review.lineageos.org" />

<remote  name="private"
       fetch="ssh://git@github.com" />

<remote  name="aosp"
       fetch="https://mirrors.tuna.tsinghua.edu.cn/git/AOSP"
       review="android-review.googlesource.com"
       revision="refs/tags/android-12.1.0_r22" />

<default revision="refs/heads/lineage-19.1"
       remote="github"
       sync-c="true"
       sync-j="8" />

<superproject name="platform/superproject" remote="aosp" revision="android-12.1.0_r22" />
<contactinfo bugurl="go/repo-bug" />
```

然后就可以`repo sync`进行同步代码了，下载完成后我们使用 AOSP 的环境设置脚本来设置环境

```bash
source build/envsetup.sh
breakfast dipper
```

下一步就出现了不一样的点，就开始正式的寻找内容了，对于下面那个可以直接跳过了，我三部设备都没用成，都是自己直接克隆东西下来，并且一加 10pro 也没人维护，更无从谈起了，没办法只能自己去下载配置和内核了

### 1. 设备

这个设备就是手机是什么型号，CPU 是什么的，主要是看 CPU 型号来进行克隆

- 小米

```text
git clone https://github.com/LineageOS/android_device_xiaomi_dipper -b lineage-19.1 dipper
git clone https://github.com/LineageOS/android_device_xiaomi_sdm845-common.git -b lineage-19.1 sdm845-common
```

按照小米的思路我们也可以寻找一加设备的内容

- 一加

```
这里缺少一加10Pro的专属平台 codename:ne2215
https://github.com/pjgowtham/android_device_oneplus_sm8450-common.git -b lineage-22.2 sm8450-common
```

然后我们来寻找内核

- 小米

```text
git clone https://github.com/LineageOS/android_kernel_xiaomi_sdm845.git -b lineage-19.1 sdm845
```

- 一加

```
https://github.com/LineageOS/android_kernel_oneplus_sm8450 -b 22.2 sm8450
```

然后来看硬件

- 小米

```
git clone https://github.com/LineageOS/android_hardware_xiaomi.git -b lineage-19.1 xiaomi
```

- 一加

```
https://github.com/LineageOS/android_hardware_oplus.git -b lineage-22.2 oplus
```
