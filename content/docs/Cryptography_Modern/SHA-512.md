SHA-512属于sha2系列，工作在64位一个字上  

1. 消息填充(Padding)
2. 将消息划分为 1024-bit block
3. 初始化 8 个 64-bit 状态变量(H0～H7)
4. 对每一个 block 进行 80 轮压缩(Compression Round)
5. 输出最终的 512-bit digest

SHA-2基本都是这个流程，SHA3可能会不一样  

### 1. 消息填充
SHA-512的填充长度是最长的，1024bit数据记一块数据,前面md5,sha1,sha256都是512bit一组  
- 原始消息后加上 0x80(即 1 bit)
- 再补若干 0 bit，直至最后 128 bit 留给“消息长度”
- 最后追加一个 128 bit 的 big-endian 数，表示消息的 bit 长
```text
message || 1 || 0...0 || (message_length_in_bits on 128 bits)
```
这个填充也就是标准流程了

### 2. 初始化hash值
SHA-512 使用 8 个 64-bit 常量作为初始状态  
这个长度也是最长的，汇编可能会使用多次左移运算来拼接

```text
H0 = 0x6a09e667f3bcc908
H1 = 0xbb67ae8584caa73b
H2 = 0x3c6ef372fe94f82b
H3 = 0xa54ff53a5f1d36f1
H4 = 0x510e527fade682d1
H5 = 0x9b05688c2b3e6c1f
H6 = 0x1f83d9abfb41bd6b
H7 = 0x5be0cd19137e2179
```

### 3. 压缩函数
SHA-512 每个 block 需要做 80 轮（round） 计算
- 一个 64-bit 的 round constant: K[i]
- 一个 64-bit 的消息调度字: W[i]
#### 3.1 σ 函数（小 sigma）
```text
σ0(x) = ROTR(x,1)  ^ ROTR(x,8)  ^ (x >> 7)
σ1(x) = ROTR(x,19) ^ ROTR(x,61) ^ (x >> 6)
```


#### 3.2 Σ 函数（大 Sigma）
```text
Σ0(x) = ROTR(x,28) ^ ROTR(x,34) ^ ROTR(x,39)
Σ1(x) = ROTR(x,14) ^ ROTR(x,18) ^ ROTR(x,41)
```
#### 3.3 CH 和 MAJ
布尔函数（选择、主元）：
```text
Ch(x,y,z)  = (x & y) ^ (~x & z)
Maj(x,y,z) = (x & y) ^ (x & z) ^ (y & z)
```

### 4. 消息调度（Message Schedule）

SHA-512 的 W 数组大小为 80 个 64-bit word  
初始化前 16 个:
```text
for i = 0..15:
    W[i] = block[i]
```
后64个
```text
for i = 16..79:
    W[i] = σ1(W[i-2]) + W[i-7] + σ0(W[i-15]) + W[i-16]
```
全部对 2^64 取模(自然溢出即可)

### 5. 每轮更新阶段变量(a,b,c,d,e,f,g,h)
初始状态
```text
a=H0
b=H1
c=H2
d=H3
e=H4
f=H5
g=H6
h=H7
```

每轮计算
```text
T1 = h + Σ1(e) + Ch(e,f,g) + K[i] + W[i]
T2 = Σ0(a) + Maj(a,b,c)

h = g
g = f
f = e
e = d + T1
d = c
c = b
b = a
a = T1 + T2
```

### 6. 最终结果回写
```text
H0 = H0 + a
...
H7 = H7 + h
```

最终输出 H0||H1||...||H7 即为 512-bit 哈希

uint64_t W[80];

//  compression function
void sha512_compress(uint64_t *state, const uint8_t *block) 
{
    int t;
    uint64_t temp1, temp2, maj, ch;

    // Prepare message 
    for (t = 0; t < 16; t++) 
	{
        W[t] = ((uint64_t)block[t * 8] << 56) | ((uint64_t)block[t * 8 + 1] << 48) |
               ((uint64_t)block[t * 8 + 2] << 40) | ((uint64_t)block[t * 8 + 3] << 32) |
               ((uint64_t)block[t * 8 + 4] << 24) | ((uint64_t)block[t * 8 + 5] << 16) |
               ((uint64_t)block[t * 8 + 6] << 8) | ((uint64_t)block[t * 8 + 7]);
    }

    for (t = 16; t < 80; t++)
	{
        W[t] = sigma1(W[t - 2]) + W[t - 7] + sigma0(W[t - 15]) + W[t - 16];
    }

    // Initialize working variables
    uint64_t a = state[0];
    uint64_t b = state[1];
    uint64_t c = state[2];
    uint64_t d = state[3];
    uint64_t e = state[4];
    uint64_t f = state[5];
    uint64_t g = state[6];
    uint64_t h = state[7];

    for (t = 0; t < 80; t++)

        b = a;

    state[1] += b;
    state[2] += c;
    state[3] += d;
    state[4] += e;
    state[5] += f;
    state[6] += g;
    state[7] += h;
}

//  final hash calculation
void sha512(const char *message, int round) 
{
    uint8_t block[128];
    uint64_t bitlen = strlen(message) * 8;
    uint64_t state[8];
    int i, len, offset;

    // Initialize hash values
    for (i = 0; i < 8; i++) 
	{
        state[i] = H[i];
    }

    // Process the message in blocks
    len = strlen(message);
    offset = 0;

    while (offset < len) 
	{
        int block_size = (len - offset < 128) ? len - offset : 128;

        // Copy the block 
        memset(block, 0, 128);
        memcpy(block, message + offset, block_size);

        // Padding
        if (block_size < 128) 
		{
            block[block_size] = 0x80;
            if (block_size < 112) 
			{
                // Append the bit length to the end of the message
                for (i = 0; i < 8; i++)
				{
                    block[112 + i] = (bitlen >> ((7 - i) * 8)) & 0xFF;
                }
            }
        }
        // Process the block
        sha512_compress(state, block);

        // Move to the next block
        offset += block_size;
    }

    // Display intermediate round values
    printf("values for round %d:\n", round);
    for (i = 0; i < round; i++) 
	{
        printf("H[%d] = %016lx\n", i, state[i]);
    }

    // Display the final hash value
    printf("Final hash value:\n");
    for (i = 0; i < round; i++) 
	{
        printf("%016lx", state[i]);
    }
    printf("\n");
}
