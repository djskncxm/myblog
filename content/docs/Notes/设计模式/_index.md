| 16  | Template Method 模板方法        | 行为型 | 框架结构        |
| 17  | Iterator 迭代器                | 行为型 | 遍历抽象        |
| 18  | Command 命令                  | 行为型 | 请求封装        |
| 19  | State 状态                    | 行为型 | 状态驱动行为      |
| 20  | Chain of Responsibility 责任链 | 行为型 | 请求传递        |
| 21  | Mediator 中介者                | 行为型 | 对象协作解耦      |
| 22  | Memento 备忘录                 | 行为型 | 状态恢复        |
| 23  | Visitor 访问者                 | 行为型 | 数据结构与操作分离   |
