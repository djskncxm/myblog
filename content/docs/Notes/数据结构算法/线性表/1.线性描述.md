### 1. 数据对象与数据结构

- 数据对象
  > 数据对象是性质相同的数据元素的集合，是一个抽象的逻辑集合。
- 数据结构
  > 数据结构是在数据对象的基础上，定义数据元素之间特定关系所形成的整体。

### 2. 线性表

线性表是具有一对一线性关系的数据元素的有限序列，它的每个实例都是元素的一个有序集合，每个实例都以有序性进行排列比如自然数，接下来我们来看线性表应该会有的一些操作，也就是我们规定数据之间的联系

1. 创建一个线性表
2. 撤销一个线性表
3. 检查是否为空
4. 确定当前表的长度
5. 按一个索引进行查找
6. 按元素进行查找对应索引
7. 按索引添加或删除某元素
8. 从头到尾输出元素

#### 2.1 抽象数据类型

> **抽象数据类型（Abstract Data Type, ADT）**  
> 是对“数据 + 能做什么操作”的一种逻辑描述，而不关心它是怎么实现的。

它定义了一组数据对象以及在这些对象上可以进行的操作，并不规定怎么实现出来，我们来看线性表的 ADT 描述

```text
实例
    有限个元素的有序表（线性序列）

操作
    判断是否为空
    返回表的长度
    返回指定索引的元素
    返回元素 x 第一次出现的位置，若不存在返回 -1
    删除索引为 index 的元素
    在 index 位置插入元素 x
    按顺序输出表中元素
```

#### 2.2 抽象类

> **抽象类（Abstract Class）** 是一个不能被实例化的类，它定义了一组必须由子类实现的接口规范。

也就是我们使用抽象类先对 ADT 进行首次实现，然后我们再进行具体类的声明和实现

```cpp
template <class T>
class linearList
{
    public:
	virtual ~linearList() {};
	virtual int size() const			                   = 0;
	virtual const T &get(int theIndex) const		       = 0;
	virtual int indexOf(const T &theElement) const	       = 0;
	virtual void erase(int theIndex)		               = 0;
	virtual void insert(int theIndex, const T &theElement) = 0;
	virtual void output(std::ostream &out) const	       = 0;
	virtual bool empty() const				               = 0;
};
```

> 只要类中包含至少一个纯虚函数（=0），该类就是抽象类
